Code Test Carousel

I prefer not to reinvent the wheel and would use one of the very many open source solution that best fits my requirements, but if I had to roll my own I would start with something like this and then extend as needed.

I would also build the project and manage dependencies with grunt or gulp and use less or sass for css. I also went with a more dynamic responsive aproach rather then making slides smaller while keeping the same number of them per view. At any rate, this is just a quick POC and by no means a robust implementation; will probably blow up in IE, works in Chrome and FF.
