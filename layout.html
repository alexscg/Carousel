Code Test Carousel

I prefer not to reinvent the wheel and would use one of the very many open source solutions that best fits my requirements, but if I had to roll my own I would start with something like this and then extend as needed.

I would also build the project and manage dependencies with grunt or gulp and use less or sass for css and templated the slides. I also went with a more dynamic responsive aproach, rather then making slides smaller while keeping the same number I keep their size constant and reduce count (better for mobile). At any rate, this is just a quick POC and by no means a robust implementation; fonts blow up in IE tltf, works fine in Chrome and FF.
